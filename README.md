## How to install
### Nuget
Use the nuget package manager console.

    `Install-Package EliteAPI`

### Nuget package manager
It is also available in the nuget package manager via Visual Studio.

Simply search for ***EliteAPI***.

### Github
You can find the latest release [here](https://github.com/Somfic/EliteAPI/releases).
