What is EliteAPI?

EliteAPI's goal is to greatly reduce the time it takes for programmers to hook into the game by doing all the parsing and deserializing of the Elite: Dangerous log files for you, leaving you to fully focus on your application.
However, EliteAPI is not soley for programmers, it also comes as a VoiceAttack and VoiceMacro plugin.

EliteAPI currently supports well over 180 events, as well as real time tracking of things like the landing gear, cargo scoop, night vision, mass lock, etc. Allare available in VoiceAttack and VoiceMacro from the get-go. EliteAPI also comes with an Inara API and Discord Rich Presence client built in. EliteAPI comes as both a .NET library as a VoiceAttack & VoiceMacro plugin, giving you a lot of possibilities to create tools you otherwise would need months for to create.
The project is completely open-source and is hosted on GitHub. The .NET library currently has more than 3,000 downloads on NuGet already!
