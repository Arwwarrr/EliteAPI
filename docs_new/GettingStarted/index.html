Getting started

A guide on how to get EliteAPI up and running.

.NET Framework

What you'll need

  • EliteAPI
  • Visual Studio
  • Elite: Dangerous
  • Basic understanding of a .NET language, such as C#

Installation

There are a number of ways to install EliteAPI. The most obvious one is using the NuGet package manager that is built-in to Visual Studio. Simply search for EliteAPI and hit install. This way updates can easily be installed later on, and all dependecies are automatically installed.

In case that's not an option, there are also other ways to install EliteAPI.

Platform Method Execution
NuGet Package Manager Install-Package EliteAPI
NuGet .NET CLI dotnet add package EliteAPI
Visual Studio By reference Download EliteAPI.dll a reference it in your project
Visual Studio By compiling Download the source code and compile the project to retrieve the EliteAPI.dll file

Starting EliteAPI

Now comes the fun part, running EliteAPI.

Once you've referenced EliteAPI to your project, add the EliteAPI namespace. Now, you're able to create an EliteDangerousAPI object, and run its Start() method. Enable logging before starting the API by calling the UseConsole() method in the Logger poperty. Congratulations, you've successfully started your first EliteAPI instance!


using EliteAPI;

namespace Example
{
    class Program
    {
        static EliteDangerousAPI EliteAPI;

        static void Main(string[] args)
        {
            EliteAPI = new EliteDangerousAPI();
            EliteAPI.Logger.UseConsole();
            EliteAPI.Start();
        }
    }
}

See for more examples and information the .NET documentation page.
