EliteVA.

VoiceAttack tokens.

EliteVA can trigger Elite: Dangerous events by triggering special commands in your VAP. EliteVA enables you to use as good as all of the events in Elite:Dangerous which can tell you a lot about what is going on in the game. You can find the full list of available events further down.

The standard syntax for events in Voice Attack is as follows: ((EliteAPI.[EventName]))}. So, if we have an event with the name EjectCargo, which will get triggered every time we eject our cargo, it'd be ((EliteAPI.EjectCargo)). To set up an event simply put create a new command and enter the name, now, every time you eject cargo in the game the command ((EliteAPI.EjectCargo)) will be ran in VoiceAttack. If you'd like an example of events, head to the examples tab.
