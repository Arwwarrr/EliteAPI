VoiceAttack tokens.

VoiceAttack uses variables in the form of tokens. EliteVA enables you to use a number of tokens which can tell you a lot about the current state of your ship and the game. You can find the full list of available tokens further down.

The standard syntax for tokens in Voice Attack is as follows: {[Type]:[Name]}. So, if we have an interger token with the name Cargo, that holds the amount of cargo we have in our ship, it'd be {INT:Cargo}. You can use these everywhere, from Voice Attack's text-to-speech feature to conditions in commands. If you'd like an example of tokens, head to the examples tab.

The following tokens are supported in VoiceAttack:

Token Explanation
{BOOL:EliteAPI.Docked} This will return true when the player is docked at a station.
{BOOL:EliteAPI.Landed} This will return true when the player is currently landed on a landable planet.
{BOOL:EliteAPI.Gear} This token will return true whenever the landing gear has been lowered.
{BOOL:EliteAPI.Shields} Returns true only if the ship's shields are online. Note that this is only true or false, and not a percentage.
{BOOL:EliteAPI.Supercruise} Returns true when in supercruise.
{BOOL:EliteAPI.FlightAssist} Will return true if the ship has flight assist enabled.
{BOOL:EliteAPI.Hardpoints} Returns true if the ship's hardpoints (or weapons) are deployed.
{BOOL:EliteAPI.Winging} If the player is currently in a wing, this token will return true.
{BOOL:EliteAPI.Lights} When the ship's lights are turned on, this will return true.
{BOOL:EliteAPI.CargoScoop} Returns true whenever the cargo scoop is open.
{BOOL:EliteAPI.SilentRunning} If the Commander has silent running enabled, this will return true.
{BOOL:EliteAPI.Scooping} Returns true whenever the ship is scooping fuel from a star.
{BOOL:EliteAPI.SrvHandbreak} This token returns true whenever the handbreak on the SRV is being used.
{BOOL:EliteAPI.SrvTurrent} If the turrent is being controlled by the player, this token will return true.
{BOOL:EliteAPI.SrvNearShip} Returns true when the SRV turrent cannot be deployed, thus the SRV being close to the ship.
{BOOL:EliteAPI.SrvDriveAssist} Returns true whenever the player has drive assist enabled in their SRV.
{BOOL:EliteAPI.MassLocked} If the ship is mass locked, this will return true.
{BOOL:EliteAPI.FsdCharging} If the frameshiftdrive is charging up for a jump, this will return true.
{BOOL:EliteAPI.FsdCooldow} Returns true when the frameshiftdrive is cooling down.
{BOOL:EliteAPI.LowFuel} If the ship is low on fuel (<25%), this token will say true.
{BOOL:EliteAPI.Overheating} If the ship's overheating this will return true.
{BOOL:EliteAPI.HasLatLong} Will return true when latitude and longitude on a planet are available.
{BOOL:EliteAPI.InDanger} Returns true if the player is in combat/danger.
{BOOL:EliteAPI.InInterdiction} If the player is being interdicted, this will return true. It will not return true in the case of the player interdicting another ship.
{BOOL:EliteAPI.InMothership} If the player is in the mothership, and thus not in a fighter/srv, this will return true.
{BOOL:EliteAPI.InFighter} Returns true when the player is in a fighter.
{BOOL:EliteAPI.InSRV} Returns true when the player is in a SRV.
{BOOL:EliteAPI.AnalysisMode} Returns true if the ship's current mode is in analysis mode, and thus not in combat mode.
{BOOL:EliteAPI.NightVision} Returns true when night vision is enabled. (Currently bugged, Frontier is working on it)
{INT:EliteAPI.Pips.Systems} Returns a number [0-8] for the amount of pips designated for systems/shields.
{INT:EliteAPI.Pips.Engines} Returns a number [0-8] for the amount of pips designated for the engines.
{INT:EliteAPI.Pips.Weapons} Returns a number [0-8] for the amount of pips designated for weapons/hardpoints.
{INT:EliteAPI.Firegroup} Returns the current firegroup.
{INT:EliteAPI.GuiFocus} Returns the currently focussed panel.
{DEC:EliteAPI.Fuel} The amount of fuel the ship currently has.
{DEC:EliteAPI.FuelReservoir} The amount of fuel the ship currently has in its reservoir.
{DEC:EliteAPI.Cargo} The number of tonnes of cargo currently being transported by the ship (INT:EliteAPI.Cargo in 2.0.0.0).
