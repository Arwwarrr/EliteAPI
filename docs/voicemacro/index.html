What is VoiceMacro?

The advanced Speech Recognition enabled Macro Software

EliteAPI only works on VoiceMacro 1.3.0.19+

VoiceMacro is a voice-controlled macro management software that let's you use your voice to control your keyboard and mouse. It can be used to control games, such as Elite: Dangerous. That is where EliteAPI comes in.

EliteAPI is not only a .NET framework library. It can also function as a VoiceMacro plugin, codename EliteMacro. EliteMacro has most of the functionalities the .NET framework has.

Getting started

Gets you started with using EliteMacro

[1]What you'll need
  • VoiceMacro
  • Elite: Dangerous
  • EliteAPI
[2]Installing the EliteMacro plugin

Installation is pretty simple. In the release tab on GitHub you'll find the EliteMacro-setup.exe file. Run it, and it will install EliteAPI to the default VoiceMacro installation directory.

There's no need to change any VoiceMacro settings once you've installed EliteMacro. If VoiceMacro was already running, restart it. Now, when you open VoiceMacro, you should see EliteMacro's log messages appear, along with a green message that reads 'EliteAPI is ready'. If you encounter any errors, join our Discord server for help.
