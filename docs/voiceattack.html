What is VoiceAttack?

Voice-activated control for your PC games and apps

VoiceAttack is a voice-controlled macro management software that let's you use your voice to control your keyboard and mouse. It can be used to control games, such as Elite: Dangerous. That is where EliteAPI comes in.

EliteAPI is not only a .NET framework library. It can also function as a VoiceAttack plugin, codename EliteVA. EliteVA has most of the functionalities the .NET framework has.

Getting started

Gets you started with using EliteVA

[1]What you'll need
  • VoiceAttack
  • Elite: Dangerous
  • EliteAPI
[2]Installing the EliteVA plugin

Installation is pretty simple. In the release tab on GitHub you'll find the EliteVA-setup.exe file. Run it, and it will install EliteAPI to the default VoiceAttack installation directory. If you are using a custom VoiceAttack installation, such as the Steam install, make sure to target the apps folder within the VoiceAttack directory for installation.

Once you have installed EliteVA, enable plugin support in the VoiceAttack settings, to make sure EliteVA can start. If VoiceAttack was already running, restart it. Now, when you open VoiceAttack, you should see EliteVA's log messages appear, along with a green message that reads 'EliteAPI is ready'. If you encounter any errors, join our Discord server for help.
