Getting started.

Installation.

There are a number of ways to install EliteAPI. These are the most used options:

Platform Method Execution
Nuget Package manager Install-Package EliteAPI
Nuget .NET CLI dotnet add package EliteAPI
Visual Studio Nuget package manager Search for EliteAPI in the package manager and install.
Visual Studio By reference Download EliteAPI.dll and reference it in your Visual Studio project.
Visual Studio Download & build Download the source code, open it in Visual Studio and build it to recieve the .dll files.
VoiceAttack Download the installer Download EliteVA-setup.exe and run it to install EliteAPI directly into VoiceAttack.
VoiceMacro Download the installer Download EliteMacro-setup.exe and run it to install EliteAPI directly into VoiceMacro.

EliteMacro only works on VoiceMacro v1.3.0.11+!

Make sure you are installing into the correct directory when installing EliteAPI for third party plugins.

Installation type Path
VoiceAttack, usual installation C:\Program Files (x86)\VoiceAttack\Apps
VoiceAttack, Steam installation C:\Program Files (x86)\Steam\steamapps\common\VoiceAttack\Apps\EliteVA
VoiceMacro C:\Program Files (x86)\VoiceMacro\Plugins

You'll have to enable Plugin Support in VoiceAttack in order for plugins to be, well, supported. After having enabled this option, you have to restart VoiceAttack.

If you have a custom Player Journal directory you can make use of the EliteVA.ini or EliteMacro.ini file, which you'll be able to find in your third-party directory after having ran EliteAPI at least once. Open the file, and write path=Your\Custom\Path\Here in it. Do not surround the path by " or ', spaces in the path are supported just fine. The next time the plugin starts up, it wil detect your custom path and look for the Journals there, if it cannot find them there, it will default back to the standard location.
