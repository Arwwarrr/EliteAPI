Getting started.

Installation.

There are a number of ways to install EliteAPI. These are the most used options:

Platform Method Execution
Nuget Package manager Install-Package EliteAPI
Nuget .NET CLI dotnet add package EliteAPI
Visual Studio Nuget package manager Search for EliteAPI in the package manager and install.
Visual Studio By reference Download EliteAPI.dll and reference it in your Visual Studio project.
Visual Studio Download & build Download the source code, open it in Visual Studio and build it to recieve the .dll files.
VoiceAttack Download the installer Download EliteVA-setup.exe and run it to install EliteAPI directly into VoiceAttack.

Make sure you are installing into the correct directory when installing EliteAPI for VoiceAttack:

Installation type Path
Usual installation C:\Program Files (x86)\VoiceAttack\Apps
Steam installation C:\Program Files (x86)\Steam\steamapps\common\VoiceAttack\Apps

You'll also have to enable Plugin Support in VoiceAttack in order for plugins to be, well, supported. After having enabled this option, you have to restart VoiceAttack.

If you have a custom Player Journal directory you can make use of the EliteVA.ini file, which you'll be able to find in your VoiceAttack directory after having ran EliteVA. Open the file, and write path=Your\Custom\Path\Here in it. Do not surround the path by " or ', spaces in the path are supported just fine. The next time EliteVA gets started up, it wil detect your custom path and look for the Journals there, if it cannot find them there, it will default back to the standard location.

If you want to update the custom path without restarting VoiceAttack, that is possible too. Create a new command, and add an Execute an External Plugin Function block. Select EliteAPI and in the plugin context panel, enter 'updatejournal'. When you now trigger this command, EliteVA will open EliteVA.ini again, and look in the custom path for Journals.
